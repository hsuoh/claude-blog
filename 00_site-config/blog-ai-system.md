## 前提：このサイトの設計思想

- **サイト名**: 薬剤師キャリアナビ（pharmirai.com）
- **コンセプト**: 転職を煽らない。データから判断させる。
- **最終ゴール**: 収益構造を理解し、合理的に転職メリットがある薬剤師を転職サイトにつなぐ
- **ターゲット**: 20〜30代の病院・調剤薬局・ドラッグストア薬剤師
- **運営体制**: PM（あなた）＋ 記事制作パートナー＋ Claude（AI担当）
- **参考設計**: リベ大ブログ教室（収益化設計）× トピッククラスター戦略（SEO構造）

## ① AIチーム構成

このブログの制作はAIと人間のハイブリッドチームで運営する。

| 役割 | 担当 | 主な責務 |
|---|---|---|
| **PM** | あなた（人間） | 方針決定・記事承認・公開判断・最終品質管理 |
| **リサーチAI** | Claude | キーワード調査・競合分析・ニーズ収集・データ収集 |
| **ライターAI** | Claude | 構成作成・本文執筆・見出し設計・ディスクリプション生成 |
| **SEO設計AI** | Claude | 内部リンク設計・カテゴリ分類・収益導線設計 |
| **データ分析AI** | Claude | 決算データ整理・調剤報酬改定解説・業界ニュースまとめ |
| **WP入稿担当** | パートナー（人間） | WordPressへの入稿・装飾・アイキャッチ作成・公開設定 |

**運用の基本原則**:
1. 記事1本につき「リサーチ → 構成提案 → 執筆 → 内部リンク設計」をClaudeが一気通貫で行う
2. PMが方向性・KW・タイトルを承認してから執筆に入る
3. 執筆後は必ずPMが一次レビュー → パートナーが入稿

## ② 役割定義

### PM（プロジェクトマネージャー）の役割
- 月次の記事テーマ優先順位を決定する
- AIが出した構成案・タイトル案を承認・修正する
- 現役薬剤師として専門的な事実確認を行う
- 転職サイトアフィリエイトの掲載タイミングを判断する

### Claude（リサーチAI）の役割
- Googleサジェスト・Xでの薬剤師コミュニティの声を収集する
- 調剤報酬改定の厚労省資料・業界ニュースを要約する
- 大手薬局の決算情報を整理して記事素材を作る
- 競合サイトの構成を分析してギャップを見つける

### Claude（ライターAI）の役割
- 読者が「転職すべきか・残るべきか・様子見か」を判断できる記事を書く
- 転職を煽らず、データと構造で読者が自己判断できる文章にする
- 吹き出し・ボックス・比較表を適切に使い、読みやすい構成にする
- 各記事に必ずディスクリプション・タイトルタグ候補を3案出す

## ③ 記事候補選定手順

記事候補は以下の順番で選定する。

```
STEP 1: ターゲットの悩みを3軸で整理する
  └ ① お金の悩み（給与・ボーナス・昇給が止まった）
  └ ② 職場の悩み（人間関係・残業・待遇が悪い）
  └ ③ 将来の不安（調剤報酬改定・薬剤師余剰・AI）

STEP 2: 各悩みを検索キーワードに変換する
  └ 「薬剤師 給料 下がる」「調剤薬局 将来性」など

STEP 3: フォルダ分類を決める
  └ ピラー（01）/ 通常記事（02）/ データ記事（03）/ 検索流入（05）

STEP 4: PMが優先順位を決定して月次計画に入れる
```

**選定の判断基準（優先度順）**:
1. 検索ボリューム 100〜3,000（競合が弱いミドル帯を狙う）
2. 現役薬剤師として専門性で差別化できるテーマ
3. 調剤報酬改定・給与・収益構造に関連するテーマ
4. 読者が「転職判断」につながる記事

## ④ キーワード選定手順

```
STEP 1: シードキーワードを設定する
  例）「調剤薬局 給与」「病院薬剤師 転職」「調剤報酬改定 影響」

STEP 2: Googleサジェストで関連KWを展開する
  → Claudeに「[シードKW]のサジェストと関連キーワードを展開して」と依頼

STEP 3: 競合上位記事の見出し構成を確認する
  → 上位3記事の見出しをClaudeに分析させ、カバーできていない切り口を探す

STEP 4: 検索意図を3種類に分類する
  ├ 情報収集型（know）→ 03_data / 05_stock フォルダ
  ├ 比較・判断型（do）→ 02_articles / 01_pillar フォルダ
  └ 転職意欲型（go）→ 02_articles（収益記事）

STEP 5: タイトルKWを確定してPMに承認をとる
```

**KWの選定ルール**:
- 1記事につき狙うメインKWは1つだけ（カニバリ防止）
- 共起語・LSIキーワードは本文中に自然に含める
- 「薬剤師」「調剤」「給与」「転職」「報酬改定」が含まれると強い
- 月間1,000以下のロングテールから始め、ピラー記事へ誘導する設計

## ⑥ 構成作成手順

Claudeへの依頼テンプレート（コピーして使う）:

```
以下の条件で記事構成を作成してください。

【メインKW】: [キーワード]
【フォルダ分類】: [01_pillar / 02_articles / 03_data / 05_stock]
【ターゲット読者】: [病院 / 調剤薬局 / ドラッグストア] 薬剤師
【読者の悩み】: [具体的な悩みを記入]
【記事の結論（読者に何を判断させたいか）】: [転職すべき / 残るべき / 様子見 / 知識習得]
【参考にすべきデータ・情報】: [調剤報酬改定の○○ / ○○社の決算 / など]

出力形式：
- タイトル案 3つ（メインKW含む・32文字以内）
- ディスクリプション案 2つ（120文字以内）
- H2見出し（5〜8個）
- 各H2の下にH3見出し（2〜4個）
- 想定文字数
- 収益導線の設置箇所
```

**構成のルール**:
- 冒頭100字以内で「誰のための記事か・何がわかるか」を明示する
- 結論を先に述べてから根拠を展開する（PREP法）
- 比較表・数値データは積極的に使う
- 「転職すべきか・残るべきか・様子見か」を明示するセクションを必ず入れる

## ⑦ 記事作成手順

```
STEP 1: PMが構成を承認する

STEP 2: Claudeに執筆を依頼する
  └ 依頼テンプレート：
    「上記の構成で本文を執筆してください。
     ・文体：です・ます調
     ・読者：現役薬剤師（専門用語OK・ただし解説を添える）
     ・転職を煽らない。データと収益構造で読者が自己判断できる内容に
     ・吹き出し・注意ボックス・比較表を適切に使う
     ・各H2の最後に「→ 詳しくはこちら」形式の内部リンク設置箇所を[LINK]と表記」

STEP 3: PMが専門的事実確認・加筆修正を行う

STEP 4: パートナーがWordPressに入稿・装飾

STEP 5: PMが最終確認して公開
```

**執筆の必須チェック項目**:
- [ ] 冒頭に「この記事でわかること」（箇条書き3〜5個）
- [ ] 著者の専門性・一次情報が含まれているか
- [ ] 調剤報酬改定の情報は最新年度のものか
- [ ] 転職サイトCTAは記事の流れに合っているか（唐突でないか）
- [ ] メタディスクリプションが設定されているか

## ⑧ 内部リンク設計手順

**トピッククラスター構造**:

```
【ピラーページ（01_pillar）】
  ↑ 複数のクラスター記事からリンクが集まる
  ↓ 各クラスター記事へリンクを張る（本文中・自然な流れで）

【クラスターページ（02_articles / 05_stock）】
  ↑ データ記事・ニュース記事からリンクが来る
  ↓ ピラーページへ必ずリンクを張る

【データ記事（03_data）】
  ↓ 関連するピラー・通常記事へリンク
```

**内部リンクのルール**:
1. 1記事に内部リンクは3〜7本（多すぎない）
2. アンカーテキストはKWを含む自然な文章にする（「こちら」は使わない）
3. 新記事を公開するたびに、既存記事から新記事への内部リンクを追加する
4. ピラー記事は常に最多の内部リンクを受け取る状態を維持する

**内部リンク管理テーブル（03_dataフォルダに随時更新）**:

```
記事タイトル | URL | カテゴリ | リンク元 | リンク先
```

## ⑨ 収益導線設計手順

**収益の流れ**:

```
検索流入記事（05_stock）
  └→ データ・解説記事（03_data）
       └→ ピラー記事（01_pillar）
            └→ 転職判断記事（02_articles）
                 └→ 転職サイトCTA（アフィリエイト）
```

**CTAの設置ルール**:

| 記事タイプ | CTA設置場所 | CTA内容 |
|---|---|---|
| 05_stock（流入記事） | 記事末尾のみ | 「給与が妥当か気になる方へ」程度のソフトCTA |
| 03_data（データ記事） | 記事末尾＋関連リンク | 「実際の転職相談はこちら」 |
| 02_articles（判断記事） | 記事中盤＋末尾 | 「転職メリットがある場合はこちら」 |
| 01_pillar（ピラー記事） | 複数箇所 | 転職サイト比較・ランキング形式のCTA |

**CTAの鉄則（転職煽り禁止）**:
- 「今すぐ転職しよう」は書かない
- 「あなたの給与は低すぎる」は書かない
- 「合理的に転職メリットがある場合は相談してみてください」のトーンを維持する
- 転職サイトは「情報収集ツール」として紹介する（「登録するだけでも価値がある」）

**導入する転職サイト（予定）**:
- マイナビ薬剤師
- ファルマスタッフ
- 薬キャリAGENT
- リクナビNEXT

### A. リサーチ依頼

```
【リサーチ依頼】
テーマ: [テーマ]
目的: pharmirai.comの記事素材として使用
対象読者: 20〜30代の調剤薬局 / 病院 / ドラッグストア薬剤師

以下をまとめてください:
1. このテーマで薬剤師が検索しそうなKW候補（10〜15個）
2. 競合上位記事がカバーしていない切り口
3. 調剤報酬改定・給与・収益構造との関連ポイント
```

```
【構成作成依頼】
メインKW: [KW]

フォルダ: [01〜05]

```
